환영 인사

첫글이자 환영인사 컨셉으로 글을 써봤습니다, 이후에 블로그가 유명해진다면 성지순례가 될 것 같네요.

hello

서론

안녕하세요, 현재 프론트엔드 8개월차 개발자 디디라고 합니다. 본격적인 프론트엔드 공부는 2023년 2월부터 시작했으며, 10월 4일부터 현업에 뛰어들게 되었습니다. 어떻게 보면 짧은 경력이지만, 한 분야에 대해 관심이 생기면 굉장히 집요하게 파고 드는 성격이라 빠르게 성장하고 있는 것 같습니다.

이 블로그를 시작한 이유?

블로그라는 것 자체를 처음 시작한 건 중학교 시절입니다. 이전부터 저만의 생각에 대해 글을 쓰는 것을 좋아했고, 그렇게 10년 이상 글을 써오다보니 글 쓰는데는 자신감이 생기더군요! 그리고 누군가에게 도움이 되는 지식 전달 행위에 대해 굉장히 큰 성취감을 느끼는 성격이기 때문에 개발 블로그를 만들어야겠다는 생각을 했습니다.

그러면 Velog나 Medium 같이 포스팅만 신경쓰면 되는 편한 플랫폼들도 있는데 굳이 하나하나 커스터마이징 해야 하는 블로그를 왜 만들었을까요? 이유는 간단합니다, 저는 완벽주의 성향이 심해서 UI, 레이아웃 구성 이런것들이 무조건 제 마음대로 움직여야 합니다. 그리고 그 어떤 프로젝트보다 블로그를 실구현 한다는 게 더 애착이 들 것 같기도 하고 말이죠.

물론 기반은 tailwind에서 만든 template를 사용했지만, 시간이 가면 갈수록 저만의 프로젝트로 변해갈 것 같습니다. 물론 아무런 기반 코드 없이 없이 만들수도 있겟지만, 잘 만들어진 기반의 프로젝트를 커스텀해가며 얻을 수 있는 지식들도 많을 것 같아요. 좋은 방식이 있다면 제 것으로 만들어야겠죠?

Welcome!

const Welcome = (props: WelcomeProps) => {
  return <h1>Welcome to {props.name} Blog !</h1>
}

const Page = () => {
  return (
      <Welcome name="CDD" />
  )
}

제가 자주 쓰는 컨벤션으로 인사를 해봤습니다. 여러모로 가져갈 것들이 많은 글들을 작성하기 위해 노력하겠습니다.
